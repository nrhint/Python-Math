Usage of Tables

Tags for tables:

  • table: the opening tag for a table.
  • thead: header for a table.
  • tr: table row, creates a single row for the table.
  • th: for creating a table header inside a row. adding
    aditional th commands will greate mutiple colums of header data.
    You want to put all of the colums here first so you dont have to
    do lots of edditing later.
  • tbody: table body command
  • td: table data, for use in the table body.
